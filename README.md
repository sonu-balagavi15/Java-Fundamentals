# Week 4 – ATM Simulation Project

## 📖 Project Overview

This project is a console-based ATM Simulation System developed in Java. It demonstrates the use of Object-Oriented Programming (OOP), file handling, and menu-driven programming.

## ✨ Features

- Secure PIN Login
- Deposit Money
- Withdraw Money
- Check Account Balance
- Store Transaction History in a File
- Console-based User Interface

## 🛠️ Technologies Used

- Java
- Visual Studio Code
- FileWriter
- Scanner
- OOP Concepts

## 📂 Files

- Account.java
- ATM.java
- ATMSimulation.java
- transactions.txt

## ▶️ How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java ATMSimulation
```

## 📚 Concepts Used

- Classes & Objects
- Constructors
- Methods
- Encapsulation
- File Handling
- Loops
- Conditional Statements
